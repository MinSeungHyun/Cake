## SplashScreen
앱을 처음 실행하면 이 화면이 뜹니다.  
전에 로그인 한 적이 있다면 여기서 로그인을 시도하고, 
로그인에 실패하거나 로그인한 적이 없다면 로그인 화면으로 넘어갑니다.
